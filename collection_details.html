Instructions for editing the table.

Adding details for a new collection:

  1. Find an existing <tr class="collection-details">...</tr> inside of <table class="collection-details">...</table>
  2. Copy and paste to an appropriate place inside of <tbody>...</tbody> (e.g. appending to the end is usually a good spot or new stuff).
  3. Modify the contents accordingly, save, review, commit, push.

Adding details for a output file for a given collection:

  1. Find an existing <ul class="output-file-details">...</ul> inside of a <tr class="collection-detail">...</tr>
  2. Copy and paste a single pair of <li class="output-file-name">...</li><li class="output-file-desc">...</li> to an appropriate place inside of <ul class="output-file-details">...</ul> (e.g. appending to the end is usually a good spot or new stuff).
  3. Modify the contents accordingly, save, review, commit, push.
